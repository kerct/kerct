Wing Tester

In order to measure drag and lift forces on a wing easily, I developed an end-to-end system for the University of Toronto's Human Powered Vehicle Design Team. From drawing the schematic and assembling the circuit board to programming a graphical user interface, this system allows users to simply plug in the load cells and a laptop to read the values in real-time. Through the interface, users can also calibrate each load cell and export the data easily.
